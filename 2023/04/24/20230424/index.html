〇、Python编程环境

（一）常用命令

venv命令：创建虚拟环境

jupyter常用快捷键：shift + enter 运行当前cell

当你在VS Code中安装了Python插件时，以下是一些常用的Python插件相关的快捷键：

  • Ctrl+Shift+P：打开命令面板
  • Ctrl+Shift+N：新建文件
  • Ctrl+Shift+X：打开扩展面板
  • Ctrl+Shift+P -> “Python: Create Terminal”：创建一个Python终端
  • Ctrl+Shift+P -> “Python: Run Python File in Terminal”：在终端中运行当前文件
  • Ctrl+Shift+P -> “Python: Select Interpreter”：选择Python解释器
  • Ctrl+Shift+P -> “Python: Start REPL”：打开交互式Python shell
  • F5：在调试模式下启动程序
  • F9：在当前行设置调试断点
  • Shift+F9：在当前行移除调试断点
  • Ctrl+K Ctrl+C：注释所选行
  • Ctrl+K Ctrl+U：取消注释所选行
  • Alt+Shift+F：格式化当前文件
  • Ctrl+Shift+I：打开Python Interactive窗口

一、一起出发！

更多思考的空间。主动探索、思考。

（一）why？目的

找到目标，方法会随之而来。思考以外的思考，工作以外的工作

  1. 为什么学编程？

    1. 将是人人必备技能
    2. 锻炼编程思维
      1. 拆解问题
      2. 量化思维
      3. 表达和描述
    3. 语言，人与机器、人与人沟通的语言
    4. 难得的及时反馈（爽），纯粹的快乐

  2. 为什么选Python？

    1. 既可以高大上，也可以很基础
      • 数据分析、深度学习、网站后台等“正统”应用；（下面的库很多是互相联系、based on的）
      • pandas、SciPy、NumPy、Matplotlib、Keras、Tensor Flow、Orange 3、theano、Scikit-learn……《Automate te Boring Stuff with Python》……
    2. 最接近自然语言
      • 关键字和使用更接近自然语言
      • 能看懂单词，哪怕不懂Python也能懂（可读性）
    3. 众多优秀的品质（可读性好、简洁、社区支持、不依赖平台、开放的库、灵活、可视化）简单易学
    4. 流行（很重要）
    5. 容易爱上
    6. 缺点
      • 速度不快
      • 动态类型，安全性不高
      • 无法编译，部署和保护不方便

（二）what？执行

一起编程、一起成长

快速扫盲，极速上手

调试

重构、版本控制等软技能

学会更好的自学

（三）how？方法

  1. 学编程，最重要的是“编程“
    1. 入门前多敲代码，少拷贝（符号、全半角等）
    2. 多做小项目（解决小问题），化零为整需要熟练
    3. 入门后多看代码，借鉴好的风格
  2. 犯错是进步的最好机会
  3. 学编程，重点不在编，在解决问题
    1. 找到问题 👉 分析问题 👉 分解问题 👉 描述问题 👉 编写代码
    2. 编程思维
  4. 先实现，再完善
    1. 不要指望一步到位
    2. 现在就动
  5. 看资料
    1. 官方文档（源头）
    2. Think Python
    3. Fluent Python
    4. 不用刷书，”逛街心态“找启发
